# diva
Scripts to use with data from the institutional repository DiVA

DiVA is a finding tool and an institutional repository for research publications and student theses, used by universites and research institutions in Sweden. Descriptons and comments are manly in swedish.

Här samlar vi skript som kan användas med data från publikationsdatabasen DiVA. De är framtagna utifrån behov vid Södertörns högskolebibliotek, men kan vara användbara för andra.
